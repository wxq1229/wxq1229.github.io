B. Reversible Cards

将一张牌$(a,b)$抽象成无向图上的一条边$(a,b)$

对于图上每个连通块分别计算

考虑一个$V$个点$E$条边连通图（图中每个点代表一种颜色，每条边代表一张牌），显然$E \ge V-1$

  • 如果$E=V-1$，那么这是棵树，无论如何做多只能取到$V-1$中颜色

    这种情况下一种构造方案是随便选个点当树根，然后每条边取深度较深的那个点，这样树根不会被取到

  • 如果$E > V-1$，至少会有一个环，可以取到$V$种颜色

    将任意一个环上相邻两点之间的所有边全部删掉，把剩下来的当一棵树（不需要管其它环），两点中的一个当根。按照树的构造方案，最后随便让删掉的一条边选根节点即可

但是并不需要输出方案

C. Too Heavy

对于每个$i$，连一条$i \rightarrow p_i$的有向边，这张图一定由若干个环构成

交换$i$与$j$手中的baggage，相当于$i$指向$p_j$，$j$指向$p_i$。目标是让$i$指向$i$。

对于一个环，找到$a_i$最大的那个点$mx$，然后一直让$mx$和$p_{mx}$交换把$p_{mx}$踢出去即可（不需要考虑交换后$p_{mx}$会不会变得tired，因为目标已经达成了）

这样$M$个点的环要花$M-1$步，显然是最少步数

D. Orientation

对于条边$(u,v)$，如果$c_u > c_v$那么这条边只能$u \rightarrow v$。同理如果$c_u<c_v$那么这条边就只能$v \rightarrow u$

把$c_u \not= c_v$ 的这些边去掉，考虑剩下的$c_u = c_v$的边。

由于保证合法，所以对于$c_u = c_v$的边，一定存在一种定向方案使得$u$和$v$能互相到达。

换句话说剩下的无向图里每个连通块都是强联通的。

考虑一个连通块的DFS树，随便个点为根，每个点一定能通过返祖边到根节点。按DFS树上的方向定向就好了

E. Simple Math 3

答案是有限的，因为要满足$A + B\cdot i + D - 1 > A + C \cdot i$

也就是$i < \frac{D-1}{C-B}$，即$i \le \lfloor \frac{D-2}{C-B}\rfloor$，令$N = \lfloor\frac{D-2}{C-B}\rfloor$

一个区间$[l,r]$内能被$D$整除的数的个数为$\lfloor\frac{r}{D}\rfloor-\lfloor\frac{l-1}{D}\rfloor$

注意到对于$\le \lfloor\frac{D-2}{B-C}\rfloor$的$i$，一定满足$0 \le \lfloor \frac{Ci + A}{D}\rfloor-\lfloor \frac{Bi + A - 1}{D}\rfloor \le 1$

换句话说包含$D$的倍数的$i$有$\sum_{i=1}^N \lfloor \frac{Ci + A}{D}\rfloor-\lfloor \frac{Bi + A - 1}{D}\rfloor$个

答案就是$N - \sum_{i=1}^N \lfloor \frac{Ci + A}{D}\rfloor-\lfloor \frac{Bi + A - 1}{D}\rfloor$，类欧即可

如果你的板子是从$0$开始的话注意减掉，代码链接

F. Do you like query problems?

神仙题

直接求和很麻烦，考虑转化为期望。我们假设所有操作随机生成

由于期望的线性性，我们考虑分别计算每个$a_i$贡献的期望，再把它每轮的贡献分开来

令$p_i$表示$i$被区间覆盖到的概率，$q_i$表示询问时$i$产生贡献的概率。则有$p_i = \frac{i(N-i+1)}{(N+1)N/2}=\frac{2i(N-i+1)}{N(N+1)},q_i = \frac{p_i}{2M+1}$

再定义$Pr(i,j,v)$表示$j$轮过后$a_i=v$的概率，则有

有个$v$的系数并且恰好等于也不是很方便计算，我们把等于改成大于等于

那么

考虑递推计算$Pr(i,j,v)$

  • $j=0$时，显然$Pr(i,j,v)=[v=0]$

  • $j >0$时，分 类 讨 论

    化简可得$Pr(i,j,v) = \frac{1+2M-p_iM}{2M+1}Pr(i,j-1,v) + \frac{p_i(M-v)}{2M+1}$

令$k = \frac{1+2M-p_iM}{2M+1},b=\frac{p_i(M-v)}{2M+1}$

有

展开前面几项可以发现$Pr(i,j,v) = b\sum\limits_{t=0}^{j-1} k^j$，后面是个等比数列可以直接求和

但是，还需要通过$Pr(i,j,v)$计算$Ans$，有式子

为了化简的方便，我们把$Pr(i,j,v)$中的$k$记为$k_i$（$k_i = \frac{1+2M-p_iM}{2M+1}$），$b = b_v \cdot p_i$（$b_v = \frac{M-v}{2M+1}$）

那么

前面一部分直接算对于后面，我们枚举$i$，我们想快速求出$\sum\limits_{j=0}^{Q-1}\sum\limits_{t=0}^{j-1} k_i^t$

交换一下求和顺序

对于第一个$\sum$，可以等比数列求和。对于第二个

考虑计算$S = \sum_{i=0}^{n-1} i\cdot q^i$。乘上$q$可以得到$qS = \sum_{i=1}^n (i-1)q^i$，那么$(1-q)S = \sum_{i=1}^{n-1}q^i - (n-1)q^n$，等比数列求和即可
