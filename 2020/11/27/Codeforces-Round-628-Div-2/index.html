D. Ehab the Xorcist

妙妙结论题

（下面$\oplus$表示xor，$\&$表示and）

首先注意到异或与和的性质

  • 1.奇偶性一定是相同的
  • 2.异或一定小于等于和

无解当且仅当$u,v$不满足上面的条件

然后考虑构造。众所周知$x \oplus x = 0$ ，同时$u,v$奇偶性相同，于是令$a = \frac{v-u}{2}$，那么$[u,a,a]$一定是一个合法的序列

我们已经找到了一个长度为$3$的合法序列，考虑有没有长度为$2$的

假设$x \oplus y = u, x + y = v$，注意到一个柿子叫

（由于xor是不进位的加法，所以做完加法过后再把进位的减回来就是xor了）

我们惊讶的发现$x \& y$是一定的，也就是$a$

问题变为了在保证$x \& y = a$以及$x \oplus y = u$的情况下能不能构造出$[x,y]$（这时候$x+y$必然为$v$）

必要条件是$a,u$在二进制下没有同为$1$的位（因为$1 \& 1 = 1, 1 \oplus 1 = 0$），此时也能构造出一个合法序列：$[u+a,a]$（$u+a$不会发生进位了）

E. Ehab’s REAL Number Theory Problem

首先由于每个数最多只有$7$个约数，所以每个数最多只会有两个不同的质因子

把所有数的质因子的指数都对$2$取模，那么剩下的数有三种情况

  • 什么也不剩了，那么选这个数就行了$Ans = 1$

  • 只剩下一个质因子，留着

  • 剩下两个质因子，留着

然后问题就是在剩下的数中选出若干个数出来，使得乘积是完全平方数

看起来还是不好做…

注意到每个数就最多与两个质因子有关了，如果选了一个数$x$满足$x = pq$，那么必定要选一个数$y$把$p$或者$q$的坑填上，假设$y = qr$，这时候我们手里已经有了三个质数$p,q,r$，如果能再选出一个$z = rp$的话就找到了一个大小为$3$的集合

$p - q - r - p$？这是一个环==

考虑建图，将每个质数抽象成一个点，对于一个有两个质因子的数$x$，假设$x = pq$，连一条$(p,q)$的双向边，问题就变成了这张图上的最小环！ 没想到吧

对于这张图上的一个环，环上必然有一个质数$\le 1000$（环的长度至少为$2$），枚举这个质数作为起点$s$，跑BFS找环就好了

最后不要忘了考虑上只剩下一个质因子的数，代码链接

F. Ehab’s Last Theorem

令$k = \lceil \sqrt{n} \rceil$

先考虑找环，只需要遍历一遍DFS树，然后每遍历到一条返祖边，看它形成环上的节点是不是$\ge k$即可，即返祖边连接的两点在DFS树上的$depth$之差$\ge k-1$

如果没有找到合法的环，那么就说明没有这样的返祖边。由于没有重边，可以发现DFS树上每个点只有不超过$k-2$条向上连的边

在DFS树上自底向上贪心将点放到独立集内，发现每个点加入后影响的点不超过$k-2$个，这样一定能找到一个合法的独立集

遍历一下即可，代码链接
